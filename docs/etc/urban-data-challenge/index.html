Urban Data Challenge 2016応募情報

■ デモモードの使用方法

① スマートフォンアプリのインストール

ParmoSenseは以下のリンクからダウンロード可能です。
端末のスペック要件についてはこちらをご確認ください。

② デモモードの起動

インストールした ParmoSense スマートフォンアプリを起動すると、 ログイン画面が表示されます。
ログイン画面右上に「TRY DEMO」というボタンがあるので、 タップするとログインを必要とせずに「デモモード」を起動することができます。

※通常、ParmoSenseを利用するためには、ログインが必要となります。

③ 写真を撮影する

共有する情報として写真をアップロードできるようになっています。 画面下部中央の「カメラボタン」をタップするとカメラを起動できます。 共有したい写真を撮影してみましょう。
撮影した写真にはコメントを付けることができます。

④ 他のユーザが共有した情報を見る

他のユーザが共有した情報は、リアルタイムに「地図」上と「タイムライン」上に反映されます。 共有された情報の詳細を見る方法は2つあります。

  • 地図上のマーカーをタップする
  • 画面下部左の「タイムライン」ボタンをタップする

■ オープンデータ化

⑤ 情報を選別・編集する

共有された情報を使って、データ編集システム上から選別や編集することにより、より品質の高いデータを生成することが可能です。 デモモードで利用可能な機能は以下のとおりです。

  • データの取捨選択
  • タイトル・詳細説明・関連URLの設定
  • 写真の回転

また、選別・編集したデータは、右上の出力ボタンから、JSONやXML、KMLなどといったデータ形式でアウトプットすることが可能です。

→ データ編集システムへアクセスする

⑥ Web API からオープンデータを取得する

ステップ⑤ で生成したオープンデータは、 Web API 経由でも取得可能です。
それぞれのデータフォーマットで出力する方法は以下のとおりです。

  • JSON： http://sv01.ubi-lab.com/Parmo/26/opendata/?format=json
  • XML： http://sv01.ubi-lab.com/Parmo/26/opendata/?format=xml
  • KML： http://sv01.ubi-lab.com/Parmo/26/opendata/?format=kml
