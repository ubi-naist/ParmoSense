ParmoSense街にいる人が「気づき」を共有できるプラットフォーム

ParmoSenseは、街歩き・観光・地域イベントなどの参加者が、互いに「位置情報」に紐づく「情報」をリアルタイムに共有するためのプラットフォームです。

目的やイベントごとに “グループ＊1” を作ることができ、同じグループに参加している人だけで情報共有することも可能です。

参加するには、スマホアプリ「ParmoSense」を入れるだけ。 グループのQRコードを読み込むことで、様々なグループに参加することができます。

＊1 グループごとに専用ParmoSenseアプリケーションが自動構築されます。

ParmoSense の活用事例

ResearchParticipatory mobile urban sensing

ParmoSenseは、奈良先端科学技術大学院大学 情報科学研究科 ユビキタスコンピューティング研究室 スマートシティ研究グループ にて開発されている研究プラットフォームです。 都市環境に眠る様々な情報を街中にいる人々のモバイルデバイスに搭載された “センサ” から収集する「ユーザ参加型センシング」の実験を ParmoSense を通じて行っています。

□ 収集可能な情報の例

ParmoSense では、様々なセンサからデータ収集することが可能＊1であるため、 幅広い都市環境の情報を収集することが可能になっています。

  • マイク・光センサ …
    「車通り」や「夜道の明るさ」などの街の安全性情報
  • 写真・移動履歴（GPS、加速度） …
    「混雑度」や「フォトジェニックな場所」などの街の観光情報

□ ユーザ参加型センシングの魅力

ユーザ参加型センシングは、街に住む人たち自らの手で、 非常に幅広い都市環境を隅々まで集めることが可能になります。

近年では、行政に頼る文化から、 行政・市民が対等な関係で街をよくしてく「市民協働」の文化への変化が広まりつつあり、 全国各地で Fix My Street をはじめ、様々な活動が始まっています。

ParmoSense は、こうした活動において、 街の現状を知るための情報収集・可視化ツールとして活用することができます。

□ 集めたデータを活用する

収集したデータは、Web API 経由で、XMLやJSON、KMLなど、 オープンデータとして利用可能な形式として出力可能です。 マッピングパーティやFix My Streetといった地域活動を支援することができます。

＊1 9軸センサ（加速度・ジャイロ・地磁気）、方位センサ、光センサ、 GPS、BLEスキャナ、外部デバイスとの連携（ウェアラブル）
